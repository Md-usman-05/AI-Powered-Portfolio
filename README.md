## 📜 Project Overview
This is not just a static website; it is a **Next-Gen Digital Dossier** designed with a **Cyberpunk/High-Tech** aesthetic. The application features a simulated Linux boot sequence, a "sentient" AI chatbot assistant, and high-end physics-based animations to demonstrate mastery in Front-End Engineering and AI integration.

## 🛠️ The Arsenal (Tech Stack)
* **Core:** React.js, JavaScript (ES6+)
* **Styling:** Tailwind CSS (Glassmorphism & Neon effects)
* **Motion:** Framer Motion (Complex transitions) & Tilt.js (3D Physics)
* **Routing:** React Router DOM (SPA Architecture)
* **Deployment:** GitHub Pages

## 🚀 Key Features
* **🤖 AI Chatbot Assistant:** A custom rule-based NLP widget that answers questions about skills and projects with simulated typing latency.
* **💻 Terminal Boot Sequence:** A realistic preloader simulating a system startup (with duplicate-check algorithms for stability).
* **⌘ Command Palette (Ctrl+K):** A keyboard-first navigation system similar to macOS Spotlight or VS Code.
* **🧲 Magnetic Navigation:** Links "snap" to the mouse cursor using physics calculations.
* **🔓 Hacker Text Effect:** Text scrambles and decrypts (Matrix-style) upon interaction.
* **🧊 Holographic Archives:** Project cards featuring 3D parallax tilt effects and glowing borders.
